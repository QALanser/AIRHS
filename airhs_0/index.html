vAIdyan: Enhancing Emergency Response with AI/ML

vAIdyan is an AI-powered emergency assistance system designed to enhance emergency response through AI/ML guidance. It provides immediate support in emergencies, streamlines communication between helpers, hospitals, and first responders, and integrates features like live accident mapping, hospital registration, and multi-modal interaction through text, images, and audio. Its ultimate goal is to save lives by improving first aid, transportation, and emotional support in critical moments​.

02

aiRis: Elevating Conversations, One Voice at a Time

aiRis is an Android-based VoiceBot delivering real-time, emotionally intelligent, and context-aware conversations in Indian English. Using LLaMA 3.2 and faster-whisper for speech recognition, it ensures seamless, human-like interactions. The system features emotion detection, personalized responses, and Pyttsx3-powered Text-to-Speech, addressing challenges like latency and context retention. aiRis empowers businesses with intuitive, AI-driven conversational solutions.

03

LendMyCompute: Decentralized GPU Lending Platform

LendMyCompute is an innovative platform designed to enable users to lend or rent computing resources like GPUs, offering a decentralized and cost-effective alternative to traditional cloud services. The system minimizes costs by implementing per-hour pricing, making high-performance computing more accessible. Built using JavaScript and Python for robust backend and frontend integration, it employs Docker and virtual machines for efficient resource sharing and automation. This project highlights the potential of decentralized platforms in democratizing access to computing resources while ensuring seamless user experience.

04

QuizTube: Check Your Attention

This project enhances learning from YouTube videos by generating intelligent, context-aware questions based on video content. Users provide a YouTube link, and the tool analyzes the transcript using speech-to-text and NLP to extract key topics. AI models then generate diverse question types to test understanding. Features include interactive testing, personalized feedback, and an intuitive interface to support active engagement and comprehension improvement.

Manikutty: Personal Routine Tracker

Manikutty is a personal routine tracker app designed to help users schedule and monitor time spent on daily activities like jogging, reading, and more. The app allows users to log and track their time spent on various tasks, providing insightful data analysis for improved productivity. Built using Google Scripts and Google Sheets as the backend, Manikutty offers seamless scheduling, real-time tracking, and comprehensive visual analytics, empowering users to optimize their routines and achieve their personal goals efficiently.
